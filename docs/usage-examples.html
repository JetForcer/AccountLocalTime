Local Time - Usage Examples

Please consider that by default widget worked only for a several US states.

1. Display the location and the current local time for Opportunity

  • 1.1. Create an account. Fill in the required fields as follows:

    • Name: 'California'
    • Billing Street: '533 Market St.'
    • Billing City: 'Fountain Valley'
    • Billing Zip/Postal Code: '92708'
    • Billing State/Province: 'California'
    • Billing Country: 'United States'

  • 1.2. Create an opportunity. Fill in the required fields as follows:

    • Opportunity Name: 'California'
    • Account Name: 'California'
    • Close Date: Choose any
    • Stage: Choose any

  • 1.3. Open created Opportunity with 'California' name and go to the Details tab
  • 1.4. At this step you can see Account local time:

2. Display the location and the current local time for Lead

  • 2.1. Create a lead. Fill in the required fields as follows:

    • Last Name: 'California'
    • Company: 'California'
    • Street: '533 Market St.'
    • City: 'Fountain Valley'
    • Zip/Postal Code: '92708'
    • State/Province: 'California'
    • Country: 'United States'

  • 2.2. Open created Lead with 'California' name and go to the Details tab
  • 2.3. At this step you can see Lead local time:

3. Use geolocation to determine local time for any town in the world

  • 3.1. Create an account. Fill in the required fields as follows:

    • Name: 'Wallenhorst'
    • Billing Street: 'Ansgarstr. 4'
    • Billing City: 'Wallenhorst'
    • Billing Zip/Postal Code: '49134'
    • Billing Country: 'Germany'

  • 3.2. Create an opportunity. Fill in the required fields as follows:

    • Opportunity Name: 'Wallenhorst'
    • Account Name: 'Wallenhorst'
    • Close Date: Choose any
    • Stage: Choose any

  • 3.3. Open created Opportunity with 'Wallenhorst' name and go to the Details tab
  • 3.4. For now the 'Account local time' field shows us: 'Unable to define time'


  • 3.5. Need to enable geolocation for this Opportunity. Go to actions and choose 'Customize local time':


  • 3.6. Activate 'Use Geolocation' checkbox and click on 'Save' button:


  • 3.7. At this step you can see Account local time:
